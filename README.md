<h4><span style='font-family:"Arial",sans-serif'>1.1 Descrição Geral do Projeto<o:p></o:p></span></h4>

<p><span style='font-family:"Arial",sans-serif'>O <span class=SpellE>TrilhaBee</span>
é um sistema web voltado para gestão, organização e monitoramento de apiários.
O sistema ajuda apicultores a substituir cadernos e planilhas por uma
plataforma mais prática para controlar colmeias, inspeções e estimativas de
produção.<o:p></o:p></span></p>

<h4><span style='font-family:"Arial",sans-serif'>1.2 Motivação e Propósito<o:p></o:p></span></h4>

informações importantes sobre as colmeias. O <span class=SpellE>TrilhaBee</span>
foi criado para facilitar esse controle, ajudando o apicultor a acompanhar
melhor a saúde das abelhas e melhorar a produção de mel com apoio da
tecnologia.<o:p></o:p></span></p>
